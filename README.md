# 勤怠管理自動化デモ（GAS）

## 概要
Google Apps Script（GAS）を使った勤怠管理業務の自動化デモです。
中小企業で多く使われているマネーフォワード クラウド勤怠のような
勤怠管理システムと連携することを想定した実装です。

## 機能
- 月末に自動実行（時間ベーストリガー）
- 従業員リストから自転車通勤者を自動抽出
- 対象者に交通費申請依頼メールを自動送信
- 個人別の申請シートをGoogleドライブに自動生成
- エラー発生時に管理者へ通知
- 外部APIからのデータ取得・スプレッドシートへの書き込み

## ファイル構成
├── common.js          # 共通設定（管理者メール・テンプレートID）
├── extractCyclists.js # メイン処理（抽出・メール送信・シート生成）
├── apiTest.js         # 外部API連携デモ
└── appsscript.json    # GASプロジェクト設定

## 技術スタック
- Google Apps Script（GAS）
- Gmail API
- Google Drive API
- Google Sheets API
- 外部REST API連携（JSONPlaceholder）

## 想定ユースケース
- 月次の勤怠確認依頼の自動化
- 条件に応じた従業員への個別通知
- 申請書類の自動生成・配布

## 開発背景
クラウドワークスで見かけた勤怠管理自動化案件を題材に、
GASを使った業務自動化の学習用デモとして開発しました。
28年間の業務改善経験をベースに、実務で使える自動化パターンを実装しています。
